# Lumina

**Free and Quick Remote Administration Tool for windows**

Lumina is a fast, quick and a light remote administration tool coded in python.

## Features
* Easy Controls
* Undetected Cilent
* Wide Variety of Commands
* Automated Startup
* Key Logger
* Remote Execution
* File Manager
* Encrypted Communication
